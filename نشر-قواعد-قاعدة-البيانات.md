# كيفية نشر قواعد قاعدة البيانات على Firebase

## الطريقة 1: استخدام Firebase CLI

1. تأكد من تثبيت Firebase CLI:
```bash
npm install -g firebase-tools
```

2. سجّل الدخول إلى Firebase:
```bash
firebase login
```

3. اختر المشروع:
```bash
firebase use restaurant-system-demo
```

4. انشر القواعد:
```bash
firebase deploy --only database
```

## الطريقة 2: من Firebase Console

1. افتح [Firebase Console](https://console.firebase.google.com/)
2. اختر مشروعك: `restaurant-system-demo`
3. اذهب إلى **Realtime Database** من القائمة الجانبية
4. اضغط على تبويب **Rules**
5. انسخ محتوى ملف `database.rules.json` والصقه في القواعد
6. اضغط **Publish**

## القواعد المحدثة:

```json
{
  "rules": {
    ".read": true,
    ".write": true,
    "restaurant-system": {
      "projects": {
        ".read": true,
        ".write": true,
        "$projectId": {
          ".read": true,
          ".write": true
        }
      },
      "sites": {
        ".read": true,
        ".write": true
      },
      "restaurants": {
        ".read": true,
        ".write": true
      },
      "orders": {
        ".read": true,
        ".write": true
      },
      "superAdmins": {
        ".read": true,
        ".write": true
      }
    }
  }
}
```

## ملاحظة أمنية:

⚠️ هذه القواعد تسمح بالقراءة والكتابة للجميع. في الإنتاج، يجب تقييد الصلاحيات حسب المستخدمين.
