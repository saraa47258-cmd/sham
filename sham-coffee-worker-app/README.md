# تطبيق قهوة الشام - نظام العامل

تطبيق سطح مكتب للعمال يفتح نظام الكاشير مباشرة.

## التثبيت والبناء

### 1. تثبيت Node.js
حمّل وثبّت Node.js من: https://nodejs.org/

### 2. تثبيت الحزم
```bash
cd sham-coffee-worker-app
npm install
```

### 3. تجربة التطبيق
```bash
npm start
```

### 4. بناء ملف EXE
```bash
npm run build
```

الملف سيكون في مجلد `dist`

## الاختصارات

- **F5** أو **Ctrl+R**: إعادة تحميل الصفحة
- **F11**: الشاشة الكاملة
- **Ctrl+Shift+I**: أدوات المطور
