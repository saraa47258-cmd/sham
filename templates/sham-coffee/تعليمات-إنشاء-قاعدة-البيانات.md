# تعليمات إنشاء قاعدة البيانات لقهوة الشام

## ✅ تم نشر قواعد قاعدة البيانات بنجاح!

## الخطوة 1: إنشاء قاعدة بيانات Realtime Database

1. افتح [Firebase Console](https://console.firebase.google.com/)
2. اختر مشروع **sham-coffee**
3. من القائمة الجانبية، اختر **Realtime Database**
4. اضغط على **Create Database**
5. اختر موقع قاعدة البيانات (مثلاً: `us-central1` أو `europe-west1`)
6. اختر **Start in test mode** (سنقوم بتحديث القواعد لاحقاً)
7. اضغط **Enable**

## الخطوة 2: تحديث Firebase Config

بعد إنشاء قاعدة البيانات، ستحصل على Database URL مثل:
```
https://sham-coffee-default-rtdb.firebaseio.com
```

### للحصول على Firebase Config الكامل:

1. في Firebase Console، اضغط على ⚙️ **Project Settings**
2. انزل إلى قسم **Your apps**
3. إذا كان لديك تطبيق Web، اضغط عليه
4. إذا لم يكن لديك تطبيق Web، اضغط على أيقونة الويب (</>) أو **Add app** → **Web**
5. انسخ Firebase Config

### تحديث ملف firebase-config.js:

استبدل محتوى `templates/sham-coffee/js/firebase-config.js` بـ Firebase Config الجديد

## الخطوة 3: تهيئة البيانات الافتراضية

بعد تحديث Firebase Config:

1. افتح الموقع: https://sham-coffee.web.app
2. اضغط F12 لفتح Developer Tools
3. اذهب إلى Console
4. اكتب الأمر التالي واضغط Enter:

```javascript
initializeDatabase()
```

5. ستنشئ البيانات التالية تلقائياً:
   - بيانات المطعم (اسم: قهوة الشام)
   - بيانات المدير (username: admin, password: admin123)
   - بيانات الموقع
   - تصنيفات افتراضية (الشيشة، المشروبات الساخنة، الباردة، الحلويات، المقبلات)
