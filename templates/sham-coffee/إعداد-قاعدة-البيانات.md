# إعداد قاعدة بيانات Firebase لمشروع قهوة الشام

## الخطوة 1: الحصول على Firebase Config

1. افتح [Firebase Console](https://console.firebase.google.com/)
2. اختر مشروع **sham-coffee**
3. اضغط على ⚙️ **Project Settings**
4. انزل إلى قسم **Your apps**
5. اضغط على أيقونة الويب (</>) أو **Add app** → **Web**
6. انسخ Firebase Config الذي سيظهر كالتالي:

```javascript
const firebaseConfig = {
  apiKey: "YOUR_API_KEY",
  authDomain: "sham-coffee.firebaseapp.com",
  databaseURL: "https://sham-coffee-default-rtdb.firebaseio.com",
  projectId: "sham-coffee",
  storageBucket: "sham-coffee.firebasestorage.app",
  messagingSenderId: "YOUR_MESSAGING_SENDER_ID",
  appId: "YOUR_APP_ID"
};
```

## الخطوة 2: تحديث ملف firebase-config.js

استبدل محتوى `templates/sham-coffee/js/firebase-config.js` بـ Firebase Config من الخطوة 1

## الخطوة 3: بنية قاعدة البيانات المطلوبة

قاعدة البيانات تحتاج إلى الهيكل التالي:

```
restaurant-system/
├── restaurants/
│   └── sham-coffee-1/
│       ├── name: "قهوة الشام"
│       ├── type: "cafe"
│       ├── username: "admin"
│       ├── password: "admin123"
│       └── status: "active"
│
├── workers/
│   └── sham-coffee-1/
│       └── {workerId}/
│           ├── name: "..."
│           ├── username: "..."
│           ├── password: "..."
│           └── position: "..."
│
├── menu/
│   └── sham-coffee-1/
│       └── {itemId}/
│           ├── name: "..."
│           ├── price: ...
│           ├── category: "..."
│           └── active: true
│
├── categories/
│   └── sham-coffee-1/
│       └── {categoryId}/
│           ├── name: "..."
│           └── order: ...
│
└── orders/
    └── sham-coffee-1/
        └── {orderId}/
            ├── items: [...]
            ├── total: ...
            ├── status: "..."
            └── createdAt: "..."
```

## الخطوة 4: إنشاء البيانات الافتراضية

بعد تحديث Firebase Config، افتح الموقع وسيتم إنشاء البيانات تلقائياً عند أول استخدام.
